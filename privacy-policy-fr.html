Politique de Confidentialité

Dernière mise à jour : Juin 2025

1. Introduction

TextLater l'application respecte votre vie privée et s'engage à protéger vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile.

2. Données Collectées

2.1 Données stockées localement

  • Messages programmés : Le contenu de vos messages et leur planification sont stockés uniquement sur votre appareil
  • Contacts : Les informations de contact sont stockées localement et ne sont jamais transmises
  • Préférences : Vos paramètres d'application (langue, format de date, etc.)
  • Templates : Vos modèles de messages personnalisés

2.2 Données d'utilisation (optionnelles)

Avec votre consentement explicite, nous pouvons collecter :

  • Données d'utilisation anonymes des fonctionnalités
  • Rapports d'erreurs techniques (sans contenu personnel)
  • Statistiques d'utilisation pour améliorer l'application

3. Utilisation des Données

Nous utilisons vos données uniquement pour :

  • Faire fonctionner l'application selon vos instructions
  • Améliorer les performances et la stabilité
  • Développer de nouvelles fonctionnalités
  • Fournir un support technique si nécessaire

4. Partage des Données

Nous ne vendons, ne louons ni ne partageons vos données personnelles avec des tiers.

Vos messages et contacts restent strictement privés et ne quittent jamais votre appareil, sauf lorsque vous choisissez explicitement de les envoyer via SMS, email ou WhatsApp.

5. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Stockage local sécurisé sur votre appareil
  • Pas de transmission de données sensibles
  • Chiffrement des données d'utilisation anonymes

6. Vos Droits

Vous avez le droit de :

  • Refuser ou retirer votre consentement pour les données d'utilisation
  • Supprimer toutes vos données en désinstallant l'application
  • Demander des informations sur les données collectées
  • Exporter vos données personnelles

7. Services Tiers

L'application peut utiliser :

  • Services système : SMS, Email, WhatsApp (selon votre choix)
  • Analytics : Données anonymes pour améliorer l'application (avec votre consentement)

8. Conservation des Données

Vos données personnelles sont conservées :

  • Localement sur votre appareil jusqu'à ce que vous les supprimiez
  • Les données d'utilisation anonymes peuvent être conservées pour améliorer l'application

9. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications importantes vous seront notifiées via l'application.

10. Contact

Pour toute question concernant cette politique de confidentialité :

Application : Utilisez la fonction "Support" dans les paramètres
